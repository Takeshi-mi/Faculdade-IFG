Cores RGB

Usamos o sistema hexadecimal. Usamos dois digitos para cada cor.

Imagine que há um balde de tinta vermelho, outro verde e outro azul. Se você misturar as tintas na proporção certa conseguirá todas as cores que desejar

#FFFFFF

F é o máximo. Se eu quiser a cor vermelha pura tenho que colocar #FF0000 = ◙ ╔▀╦

Mexendo com LISTAS

1. Lista não ordenada

Não importa a ordem

A tag é <ul> (undefined list) </ul>

  • arroz
  • Produtos
    • Limpeza
      • Qruim
  • pão de queijo
  • chuchu
  • abóbora

2. Lista ordenada

Neste tipo de lista a ordem importa

A tag é <ol> (ordenated list) </ol>

Para os itens a tag é <li> (list item) </li>

  1. Luiz
  2. Maz
  3. Sebastião
  4. Jorge

Para fazer provas, por exemplo:

  1. Qual é a metade de 2+2?
    1. 1
    2. 2
    3. 3
    4. 4
  2. Posso começar de qualquer letra passando o parâmetro start="5" dentro da tag <ol>
    1. 1
    2. 2
    3. 3
    4. 4
  3. Posso usar números Romanos passando o parâmetro type="i" dentro da tag <ol>
    1. 1
    2. 2
    3. 3
    4. 4
